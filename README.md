# PART 1 — PROJECT PLANNING & STRUCTURE

## 1. Project Overview
Perfect Glow is a handmade soap brand that creates natural, chemical-free skincare products.  
This project showcases a simple, clean, and responsive 5-page website using **HTML and CSS**.

The website includes:
- Home  
- About  
- Products  
- Enquiry  
- Contact  

---

## 2. Website Goals and Objectives
- Display product prices, ingredients, and product information  
- Provide excellent user experience  
- Allow easy navigation and usability  
- Build a functional and clean website  
- Increase sales and awareness through the website  

---

## 3. Key Features
- Fully responsive layout  
- Clean, simple visual design  
- Consistent navigation across all pages  
- Mobile-friendly hamburger menu  

---

## 4. Sitemap
- **index.html** – Home  
- **about.html** – About  
- **products.html** – Products  
- **enquiry.html** – Enquiry Form  
- **contact.html** – Contact Information and Form
